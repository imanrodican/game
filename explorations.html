Explorations in Gaming

Game Development

The first iteration of Retail Resort Tycoon did not include the physical components, and was intended to be played on a computer rather than on a game board. However, we quickly realized that without a physical component, the game would be less interactive between players. The game pieces additionally add excitement to the game, as players are able to design a store layout between turns.
