Game Description

Retail Resort Tycoon is a 2-player, competitive game that teaches players how to manage money and run a small-business

Game Pieces

The game includes two of each of the following pieces:

  • A game board
  • Four regular shelves
    1. A bread shelf
    2. A shampoo shelf
    3. A sunscreen shelf
    4. A towel shelf
  • Three special shelves
    1. A Christmas shelf
    2. A New Year's Shelf
    3. A Valentine's Day shelf
  • A cash register
  • 5-dollar Retail Bucks
  • 10-dollar Retail Bucks
  • 50-dollar Retail Bucks
  • 100-dollar Retail Bucks

Setting up the Game
