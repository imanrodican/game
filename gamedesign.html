Game Description

Retail Resort Tycoon is a 2-player, competitive game that teaches players how to manage money and run a small-business

Game Pieces

The game includes two of each of the following pieces: a game board, four regular shelves (a bread shelf, a shampoo shelf, a sunscreen shelf, and a towel shelf), three special shelves (a Christmas shelf, a New Year's Shelf, and a Valentine's Day shelf), and a cash register. The game also includes 5-, 10-, 50-, and 100-value Retail Bucks, the currency of Retail Resort Tycoon.
